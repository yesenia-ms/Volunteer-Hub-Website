Volunteer Hub Website made for volunteers and organizations to search and post events.

Volunteers are able to
- create accounts
- log in
- search events
- register for events
- unregister for events
- message Volunteer Hub
- message organizations

Organizations are able to 
- create accounts
- log in
- post events
- delete events
- message Volunteer Hub
